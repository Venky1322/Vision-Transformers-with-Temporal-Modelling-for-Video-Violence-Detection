# Violence Detection Using Deep Learning

## Project Overview
This project detects violent activities in videos using Deep Learning and Computer Vision techniques.

## Features
- Video violence detection
- Deep learning model training
- Real-time prediction
- Accuracy evaluation

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib

## Dataset 
Dataset source : 
https://www.kaggle.com/datasets/mohamedmustafa/real-life-violence-situations-dataset
